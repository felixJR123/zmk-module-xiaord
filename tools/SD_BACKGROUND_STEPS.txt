Xiaord SD Background Converter Steps
====================================

What this is for
----------------
Use this when you want many background pictures on the XIAO Round Display
microSD card. The tool converts JPG/PNG files into 240x240 RGB565 files named:

  bg001.rgb565
  bg002.rgb565
  bg003.rgb565

The firmware reads those converted files from:

  /SD:/xiaord-bg/converted


First-time setup from the zmk-module-xiaord repo
------------------------------------------------
1. Put the microSD card in your computer.

2. Find the SD card drive letter. On Windows it might be something like:

  E:\

3. Put the pictures you want to use in a normal folder on your computer, for example:

  C:\Pictures\xiaord

4. From the zmk-module-xiaord repo, run:

  python tools\xiaord_sd_backgrounds.py E:\ --source C:\Pictures\xiaord

If you do not want to clone the whole repo, download these three files and keep
them together in one folder:

  xiaord_sd_backgrounds.py
  convert_xiaord_bg.py
  SD_BACKGROUND_STEPS.txt

Then run from that folder:

  python xiaord_sd_backgrounds.py E:\ --source C:\Pictures\xiaord

If a path has spaces, put quotes around it:

  python xiaord_sd_backgrounds.py E:\ --source "C:\Users\fhern\Pictures\Xiaord Backgrounds"

5. The tool creates this SD card layout:

  E:\xiaord-bg\raw
  E:\xiaord-bg\converted
  E:\xiaord-bg\tools

6. Put the SD card back into the XIAO Round Display.


Adding or changing pictures later
---------------------------------
1. Put the microSD card in your computer.

2. Add or remove JPG/PNG files in:

  E:\xiaord-bg\raw

3. Run the converter that was copied onto the SD card:

  python E:\xiaord-bg\tools\xiaord_sd_backgrounds.py E:\

4. Put the SD card back into the XIAO Round Display.


Useful options
--------------
Move the crop left/right:

  --center-x 0.45

Move the crop up/down:

  --center-y 0.45

Zoom in:

  --zoom 1.20

Example:

  python E:\xiaord-bg\tools\xiaord_sd_backgrounds.py E:\ --center-x 0.50 --center-y 0.45 --zoom 1.15


Firmware config example
-----------------------
Use SD backgrounds, and fall back to BG_4 if the SD card is missing:

  CONFIG_XIAORD_BG_1=n
  CONFIG_XIAORD_BG_2=n
  CONFIG_XIAORD_BG_3=n
  CONFIG_XIAORD_BG_4=y
  CONFIG_XIAORD_BG_SD=y
  CONFIG_XIAORD_BG_SD_VOLUME_NAME="SD"

  CONFIG_XIAORD_BG_SD_ROTATE_MS=60000
  CONFIG_XIAORD_BG_SD_RETRY_MS=5000
  CONFIG_XIAORD_BG_SD_GESTURES=y

Use whichever compiled background you want as fallback. For example, set
BG_2=y instead of BG_4=y if you want BG_2 to appear when the SD card is missing.

If the card is inserted but the keyboard still uses the fallback background,
format the card as FAT32, confirm the converted files are in
xiaord-bg\converted, and keep CONFIG_XIAORD_BG_SD_VOLUME_NAME="SD" unless your
board registers the SD disk with a different name. Insert the card before the
keyboard boots.


Notes
-----
- PNG works without extra Python packages.
- JPG/JPEG requires Pillow. Install it with:

  python -m pip install pillow

- Each converted picture is 115,200 bytes.
- Files are used in numeric order: bg001, bg002, bg003, then wrap back to bg001.
- CONFIG_XIAORD_BG_SD_MAX_FILES=999 is only a maximum. It is fine to have fewer
  pictures on the SD card.
